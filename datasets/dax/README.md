# `dax` dataset

Dense **option-surface calibration** benchmark built from historical
European option quotes on the DAX index. All strikes and prices are
normalised by the spot $S_0 = 25\,280$, so $Y_0 = 1$ and
$K_{\mathrm{norm}} = K/S_0$. Maturity indices are recomputed on the
model time grid (default $T = 1$, $N = 2048$ steps).

## Strike grid

For every available maturity the loader picks evenly spaced normalised
strikes inside each band, snapped to the nearest observed strike from
the raw quote table:

| Group | # strikes | Strike range |
|-------|-----------|--------------|
| Vanilla calls | 25 | $K_{\mathrm{norm}}\in[0.88,\,1.12]$ |
| Vanilla puts  | 25 | $K_{\mathrm{norm}}\in[0.88,\,1.12]$ |
| Deep-OTM puts | 20 | $K_{\mathrm{norm}}\in[0.60,\,0.88]$ |
| Deep-OTM calls | 15 | $K_{\mathrm{norm}}\in[1.08,\,1.35]$ |
| Digital puts (put-spread synthesised) | 12 | $K_{\mathrm{norm}}\in[0.65,\,0.98]$ |

The two OTM groups and the digital puts are the **rare-event side** of
the surface — these are the targets that benefit from the Girsanov
overlay (one tilt controller per OTM side in `exp_dax`).

## Train / eval split

`exp_dax/masking.py:create_strike_masks(eval_frac=0.25, seed=0)` holds
out a quarter of the strikes in every group. Held-out strikes are not
contiguous; train and eval cover overlapping ranges so the model must
interpolate rather than extrapolate. We report train MSE, eval MSE on the
masked strikes, the eval–train gap, and eval MAE.

## Files

| File | Content |
|------|---------|
| `dax_dataset.py` | Base loader (`DaxOptionsDataset`, `load_dax_options`) |
| `dax2_dataset.py` | Dense-defaults wrapper used by v3 (`load_dax2_options`) |
| `dax_options.csv` | Raw quote table |
| `dax_dataset.pkl` | Pickled `DaxOptionsDataset` (regenerated by `prepare.py`) |
| `__init__.py` | Re-exports `DaxOptionsDataset` |

## (Re)generating

```bash
python -m v3.datasets.prepare --dax                  # default spot 25280, N=2048
python -m v3.datasets.prepare --dax --spot 24000
```

## Loading

```python
from v3.datasets.prepare import load_dax
from v3.experiments.exp_dax.masking import create_strike_masks
ds = load_dax()
masks = create_strike_masks(ds, eval_frac=0.25, seed=0)
print(ds.spot, ds.call_prices.shape, ds.otm_put_prices.shape)
```
