# `toy` dataset

A controlled synthetic benchmark for **path-functional calibration**.
Samples are scalar paths $Y_t$ on $[0, T]$ with $T = 1$, discretised on
a uniform grid of $N = 2048$ steps.

## Ground-truth SDE

The dataset is generated from a three-component non-linear Itô SDE with
*independent* diagonal diffusion:

$$
\begin{aligned}
\mathrm{d}X^1_t &= -X^1_t\big((X^1_t)^2 - 1\big)\,\mathrm{d}t
   + \sqrt{|X^1_t| + 1}\,\sqrt[3]{X^1_t + 5}\,\mathrm{d}W^1_t,
   & X^1_0 &= 0.5,\\
\mathrm{d}X^2_t &= \sin(X^2_t)\big(2 - (X^2_t)^2\big)\,\mathrm{d}t
   + \sqrt[3]{1 + (X^2_t)^2}\,\mathrm{d}W^2_t,
   & X^2_0 &= 0,\\
\mathrm{d}X^3_t &= \big(X^3_t - (X^3_t)^3/3\big)\,\mathrm{d}t
   + \exp\!\big(-(X^3_t)^2/4\big)\,X^3_t\,\mathrm{d}W^3_t,
   & X^3_0 &= 1.
\end{aligned}
$$

The observable is the linear combination

$$
Y_t = 0.40\,X^1_t + 0.35\,X^2_t + 0.25\,X^3_t,
$$

so $Y_0 = 0.45$. The SDE is simulated with Euler–Maruyama on $N$ steps
for $n_{\mathrm{mc}} = 2\times10^{5}$ Monte-Carlo paths (default).

## Supervised targets

From the simulated paths we compute four classes of targets at the
maturities $\mathcal{T} = \{0.1, 0.25, 0.5, 0.75, 1.0\}$:

* **Soft-max payoffs** at strikes
  $\mathcal{K} = \{-0.5, -0.25, 0, 0.25, 0.5, 0.75, 1.0\}$:
  $\mathbb{E}\!\big[\max(Y_T - K, 0)\big]$.
* **Running maximum**: $\mathbb{E}\!\big[\max_{0 \le t \le T} Y_t\big]$.
* **Squared path average**:
  $\mathbb{E}\!\big[\tfrac{1}{T}\!\int_0^T Y_t^2\,\mathrm{d}t\big]$.
* **Barrier hit probabilities**
  $\mathbb{P}\!\big(\max_{0 \le t \le T_i} Y_t \ge H_j\big)$ at three
  barrier levels $H_j$ taken as the $80\%$, $90\%$ and $95\%$ empirical
  quantiles of the terminal running maximum.

The barrier and the high-strike payoffs are the rare-event side of the
loss. The squared path average is the typical-path side.

## (Re)generating

```bash
python -m v3.datasets.prepare --toy            # 200k MC paths, default
python -m v3.datasets.prepare --toy --n-mc 500000
```
